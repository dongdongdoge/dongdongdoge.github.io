# Wentao Dong — Academic Website

Source for [dongdongdoge.github.io](https://dongdongdoge.github.io), a personal academic website built with Jekyll and GitHub Pages.

## Site structure

- Home and selected publications: `_pages/about.md`
- Research overview: `_pages/research.md`
- Teaching, talks, and service: `_pages/teaching.md`
- X-Crypt: `_pages/x-crypt.md`

## Local development

```bash
bundle install
./run_server.sh
```

Then open `http://127.0.0.1:4000/`.

The design retains components derived from Minimal Mistakes and AcadHomepage; their license notices remain in `LICENSE` and the relevant source files.
